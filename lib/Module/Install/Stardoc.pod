=encoding utf8

=head1 NAME

Module::Install::Stardoc - Stardoc Support for Module::Install

=head1 SYNOPSIS

In Makefile.PL:

    use inc::Module::Install;

    stardoc_make_pod;
    all_from 'lib/Foo.pm';

=head1 DESCRIPTION

This command generates a pod file from every .pm file in your lib/ directory that contains Stardoc documentation.
